# 班会登録アプリ

NPO国際ボランティア学生協会IVUSA向けに制作した、班会登録を簡潔かつ高速に行うためのWebアプリです。

## 背景

従来の班会登録では、

- 登録場所であるホームページを開くまでに時間がかかる
- 入力項目が多い
- 登録の心理的ハードルが高い
- 「あとでやろう」となり登録漏れが発生する

といった課題がありました。

また、班会の実施状況が集まっていなく、
継続率向上や早期フォローにも繋げづらい状態でした。

そこで、

「開いてすぐ登録できる」
「班会の流れの中で30秒で終わる」

ことを重視し、UI/UXを設計しました。

---

## 主な機能

- 班会登録
- 実施日時の登録（自動で当日の日付が入る）
- 参加者登録
- 返信速度（◎○△）の可視化
- 班会は異様の入力
- スマホ最適化UI
- ログイン状態保持
- カレンダー入力

---

## 工夫した点

### ① 操作数の削減

「開いてから登録まで」を最短化するために、

- モバイルUI設計（1画面で完結）
- タップ中心の入力（返信状況、内容をタップを中心に）
- ログイン保持

などを実装しました。

---

### ② 班会と登録を一体化

「登録を後回しにしない」ために、

- 班会開始直後
- もしくは班会終了直前

の30秒で登録を完了させる運用を想定しています。

単なるシステムではなく、
運用設計まで含めて考えています。

---

### ③ 継続率向上への活用

登録データを活用することで、

- 班会実施状況の可視化
- 参加傾向分析
- 早期フォロー
- 継続率向上

に繋げることを目的としています。
